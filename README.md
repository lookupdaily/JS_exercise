# JS Exercise - 1

## Task 1

Write a JavaScript function to  get the number of occurrences of each letter in specified string

```
Sample arguments : ‘test’, ’t’
Expected Output :  2
```

how it might look:
```javascript
countCharacter('test','t') // => 2
```

### Test cases
| Input | Output |
|-------|--------|
|'a','a'| 1 |
|'ab','a'|1 |
|'aab','a'|2|
|'test it', 't'|3|
|'a','b'|0|
|'','a'|0|
